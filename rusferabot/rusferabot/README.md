#  Telegram-бот для сервисного центра "Русфера"

Бот для автоматизации приёма заявок на ремонт техники и консультации клиентов.

##  Функционал
-  Оформление заявок на ремонт
-  Проверка статуса заявки
-  Просмотр услуг и цен
-  Контактная информация
-  Автоматическое обновление статусов (имитация)
-  Сохранение данных в JSON
